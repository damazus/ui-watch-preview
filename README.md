## Getting Started

These instructions will get you a copy of the project up and running on your local machine
 for development and testing purposes.


### Installing

```
git clone git@github.com:damazus/ui-watch-preview.git
```

```
cd (to project folder)
```

```
npm install
```

### Run project
```
gulp
```
